Usage with css:

Case: 

.case-small .css:after {
  content: 'Small';
}
.case-medium .css:after {
  content: 'Medium';
}
.case-wide .css:after {
  content: 'Wide';
}

Cases via javascript events

document.addEventListener('caseChange', function(event) {
    console.log('caseChange.detail', event.detail);
  });
